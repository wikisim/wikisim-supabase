# Security Policy

## Supported Versions

The content of the `main` branch of this repository will be adhoc applied to / synchronised with wikisim.org's Supabase server.
Please inform us of any vulnerabilities in this branch using the private "[Report a vulnerability](https://github.com/wikisim/wikisim-supabase/security/advisories/new)".

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |

## Reporting a Vulnerability

To report this vulnerability please use the private "[Report a vulnerability](https://github.com/wikisim/wikisim-supabase/security/advisories/new)"

Thank you for your message; there are no guaranteed response times as the team is currently one unpaid person.
If you don't receive a response in 5 days please repost.
If you think it is urgent the please repost up to every 12 hours or email us at wikisim dot secure at gmail dot com

If the vulnerability is accepted you'll get gratitude and, if you wish, a personal public thank you in the list on contributors to the code base.
